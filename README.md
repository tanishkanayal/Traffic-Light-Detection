# Traffic-Light-Detection
Overview

Traffic Light Detection is a MATLAB-based computer vision project designed to detect and classify traffic light signals in images and videos. The system identifies red, yellow, and green traffic lights using image processing techniques and color-based analysis.

Features

- Traffic Light Detection
- Red Light Recognition
- Yellow Light Recognition
- Green Light Recognition
- Real-Time Video Processing
- Color-Based Classification
- Automated Signal Analysis

Technology Stack

- MATLAB
- Image Processing Toolbox
- Computer Vision Techniques
- HSV Color Space Analysis
- Video Processing

Components Used

- MATLAB
- Input Images/Videos
- Image Processing Toolbox

Project Structure

```text
Traffic-Light-Detection-MATLAB/
├── Source_Code/
├── Sample_Videos/
├── Documentation/
├── Report/
└── Results/
```

Working Principle

1. Input video data is provided to the system.
2. Frames are processed using image processing techniques.
3. RGB images are converted to HSV color space.
4. Color thresholds are applied to identify red, yellow, and green regions.
5. Detected traffic light signals are classified and displayed.
6. The system outputs the detected traffic light status.
